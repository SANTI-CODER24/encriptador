Guia basica sobre el juego Encriptador de texto

Para crear un encriptador de texto básico, necesitas una interfaz de usuario con campos de entrada y salida, y botones para encriptar y desencriptar. Utiliza un algoritmo de encriptación sencillo como el cifrado César, donde cada letra se reemplaza por otra que se encuentra un número fijo de posiciones más adelante en el alfabeto. El proceso de encriptación implica leer el texto ingresado, aplicar el algoritmo y mostrar el texto encriptado, mientras que el proceso de desencriptación implica leer el texto encriptado, aplicar el algoritmo inverso y mostrar el texto original.
